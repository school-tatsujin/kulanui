# Kulanui

Kulanui は、AI を活用して「濃く・親切で・幅広い」学習コンテンツを作るオープンプロジェクトです。GitHub Pages で公開する Markdown 形式のテキストを中心に、専門書レベルの知識をわかりやすく整理することを目指しています。

Kulanui の特徴は、単なる「AI文章の寄せ集め」ではなく、学習者が理解しやすい構造や流れを **人間の編集者が設計し、AI による高速な執筆と組み合わせる**点にあります。記事はすべて体系化された構造の中に配置され、百科事典と教科書の中間に位置する“学習のための知識体系”として統合されています。

現在はテスト段階のため、主催者が個人で手動でコンテンツ生成をしています。将来的には、Wikipediaのような共同編集の仕組みを作るか、自動生成を行おうと思っています。プロンプト一発で安定した結果がでないのが今のところの悩みで、完全自動生成には程遠く、人間の編集者による指示とレビューが必要です。

## 目指すもの
- **コンセプト**: 百科事典並みの網羅性と、教科書並みの丁寧さを両立させる。
- **対象読者**: 大学生から専門職まで、前提知識が少なくても読めるが、さらに深く学べる構成。
- **執筆ポリシー**: 主要概念の直感的説明と厳密な定義を併記し、歴史的背景や最新動向をバランスよく紹介する。
