# Kulanui (クーラヌイ)

Kulanui は、AI を活用して「濃く・親切で・幅広い」学習コンテンツを作るオープンプロジェクトです。GitHub Pages で公開する Markdown 形式のテキストを中心に、専門書レベルの知識をわかりやすく整理することを目指しています。

- **コンセプト**: AI-powered Wikipedia plus。百科事典並みの網羅性と、教科書並みの丁寧さを両立させる。
- **対象読者**: 大学生から専門職まで、前提知識が少なくても読めるが、さらに深く学べる構成。
- **執筆ポリシー**: 主要概念の直感的説明と厳密な定義を併記し、歴史的背景や最新動向をバランスよく紹介する。
